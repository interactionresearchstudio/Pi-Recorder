# Pi Recorder

This is an openframeworks based Audio Recorder made for use with a Raspberry Pi Zero.

## Requirements

- OpenFrameworks v0.9.0 (only tested)
- ofxGPIO
- ofxMaxim (currently an edited version in the repo)
- ofxJSON
- USB Audio Interface 
- Raspberry Pi Zero (only PI tested)
- 8GB SD card
